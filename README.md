## Getting Started

# To run development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

# Start the app:

  1. Ensure that the accompanying server app is running (default localhost:5000)

  2. Build app
    ```bash
    npm run build
    ```

  3. Start
    ```bash
    npm start
    ```

  4. The app should now be running on http://localhost:3000.
